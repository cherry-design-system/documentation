**Technology:**
- ⚛ [Next.js](https://nextjs.org/)

## Install dependencies
```
npm install
```

## Runs the app in the development mode
Build and open your browser to http://localhost:3000.
```
npm run dev
```

Deploy with [Vercel](https://vercel.com/) 🚀
