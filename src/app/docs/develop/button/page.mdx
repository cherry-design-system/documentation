# Button

Buttons are essential components in web development, frequently used for user interactions. In this guide, weâ€™ll explore various button variations and their associated properties. 

Keep in mind that the **\<button>** element inherently includes native browser properties like onClick. Additionally, Cherry UI components introduce custom properties that always begin with a **$** to distinguish them from native component props.

```jsx
import React from "react";
import { Button } from "cherry-styled-components";

export default function Page() {
  return (
	<Button>Button</Button>
  )
}
```

It looks just like this:

<PreviewTile
	$bgColor="#FFFFFF"
	$title="Primary"
	$subTitle="Default"
>
	<Button>Button</Button>
</PreviewTile>

You can change the look of your button by adding a **$variant** prop.

```jsx
import React from "react";
import { Button } from "cherry-styled-components";

export default function Page() {
  return (
	<Button $variant="secondary">Button</Button>
  )
}
```
It looks just like this:

<PreviewTile
	$bgColor="#FFFFFF"
	$title="Secondary"
	$subTitle="Default"
>
	<Button $variant="secondary">Button</Button>
</PreviewTile>
